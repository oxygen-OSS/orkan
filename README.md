# Orkan
Orkan is a Free and Open Source Lua utility for installing packages on Arch-based Linux distributions in a CLI (Command-Line Interface).
The tool strives to make Arch Linux simpler to setup and more accessible, even to beginners.

![](https://raw.githubusercontent.com/oxygen-OSS/orkan/main/assets/orkan.png)

TO Run it on Arch Linux and it's flavours:

## LATEST RELEASE:
```
sudo pacman -S lua
curl -L -o orkan.lua https://raw.githubusercontent.com/oxygen-OSS/orkan/main/releases/orkan-1.1stable.lua
lua orkan.lua
```

## OLDER RELEASE:
```
sudo pacman -S lua
curl -L -o orkan.lua https://raw.githubusercontent.com/oxygen-OSS/orkan/main/releases/orkan-1.0stable.lua
lua orkan.lua
```
